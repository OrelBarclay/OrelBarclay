Elite of Durham Auto

The auto shop web system you're developing is designed to streamline operations within an auto repair shop. It features a user-friendly interface for customers to book appointments, search for services based on car make, model, and year, and manage their vehicle maintenance records. The system also includes an integrated service catalog, pricing calculator, and customer communication tools like SMS notifications. With real-time updates on service status and a secure payment gateway, this system aims to enhance customer satisfaction and improve the efficiency of shop operations.

This web solution allows administrators to have full control of the website. They are able to:

  1. add new products and services
  2. remove products or services
  3. add items to promotion
  4. set and modify prices
  5. view orders
The system alerts admins when an order has been made. It allows them to view and varify orders before they fulfill it. This is a creative web solution....
