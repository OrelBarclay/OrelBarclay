Cargo Shipping

The cargo shipping solution offers many features. One main one is that it allows customers to see the cost of pixcking up their packages which is calculated based on their location. The software also calculates the cost of shipping the item (s) for each client. Clients are able to save items in their shopping cart or pay immediately using the online payment processor. Another customer feature is the convenience of adding comments and store favorite items.

This web solution allows administrators to have full control of the website. They are able to:

  1. add new products and services
  2. remove products or services
  3. add items to promotion
  4. set and modify prices
  5. view orders
The system alerts admins when an order has been made. It allows them to view and varify orders before they fulfill it. This is a creative web solution....
